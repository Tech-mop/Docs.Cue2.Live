# Roadmap

This page outlines the upcoming planned features and improvements.
Upcoming features are in order of development. Future plans are long term visions of software capability.

## Upcoming Features

- Expanded control cues
  - Cues that control other cues
- Text output
    - Rich text with formatting
    - Multi-cue text with active-cue controls
    - Teleprompt scroll
- Disarm / Arm cues
- Integrated fades for audio
- Plug-in support
- Passive cues (conditional cues)
  - Cues that are not activated but can be conditional.
- General network cues
- User defined components
  - Can add predefined actions as a component
  - Ie projector controls
- DMX output support via sACN and Artnet
- Timecode output cue component support 
- Midi output cue component
- Expanded cue trigger support
  - Midi triggers
  - Timecode triggers
  - Hotkey triggers
- Show / Edit mode
  - Disable inspectors and cue changes in show mode
- Microphone input cue component
- Camera input cue component

## Future Plans

- Headless Build:
  - Enables use on single-board computers. In tandem with the upcoming session link feature will enable distributed processing.
  - Use case: for multiroom av deployments with central control. 
- Cloud synchronization features:
  - Store copies of your showfiles or library on the cloud for easy access and sync when moving between computers and locations.
  - Remote showfile collaboration.
- Session Remote Access:
  - Linking showfiles beyond local networks. 
- Mobile phone / tablet support:
  - Dedicated app to link with your session and provide playback controls.
